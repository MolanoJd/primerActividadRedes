5. Identificar los componentes clave de cualquier red de datos y describir su funcionamiento.
6. Responda brevemente, ¿Qué son los datos? ¿Qué es una red de datos? ¿Qué es una red convergente?

Los datos se definen como información obtenida de hechos u opiniones  que se obtienen de diversas fuentes, recopilan y se almacenan con el fin de que  pueda ser procesada y utilizada para tomar decisiones, al transmutar este concepto al campo de estudio de las redes las fuentes serían dispositivos.


 Los datos se dividen en paquetes y se envían a través de la red utilizando diferentes protocolos de red.

Cuando se habla de  red de datos  se hace referencia a un conjunto de nodos (Computadores) que haciendo uso de las telecomunicaciones que permite intercambiar datos. En las redes de cómputo, dispositivos de computación conectados en red (nodos de la red) pasan los datos entre sí a lo largo de las conexiones de datos.
 
Cuando la comunicación combina múltiples tipos de datos, como voz, video y datos, en una única red le llamamos red de convergencia. La convergencia se refiere al hecho de que estos diferentes tipos de tráfico se unen y se entregan de manera eficiente y efectiva a través de la misma red.


7. Describir las características de las arquitecturas de red: tolerante a fallas, escalabilidad, calidad del servicio y seguridad. Mostrar sus respectivos ejemplos. 
8. Investigar qué son las redes orientadas y no orientadas a la conexión.

Red orientada a la conexión: Son los dispositivos establecen una conexión dedicada, generalmente se refiere a una conexión de banda ancha que se utiliza para conectarse a Internet a través de un proveedor de servicios de Internet (ISP), antes de que se pueda transmitir información. La conexión se mantiene durante la transmisión de datos y se libera una vez que se completa la transferencia de datos. Un ejemplo se da cuando en las llamadas establece una conexión entre dos dispositivos antes de que se pueda realizar una llamada telefónica.

Las redes no orientadas: Se refieren a redes de comunicación en las que los datos enviados tienen un orden establecido o alguna jerarquía, es decir que  no están organizados en un flujo de comunicación definido, es decir, no se establece una conexión previa y no se garantiza que los paquetes de datos lleguen en un orden específico.

Preguntas de lectura: Capítulo 2
1. Describir la estructura de una red, incluidos los dispositivos, medios y servicios necesarios para lograr comunicaciones exitosas. 
2. Comparar y contrastar los siguientes términos: RED, LAN, WAN, INTERNETWORK E INTERNET.

Anteriormente ya definimos lo que es la red, conjunto de dispositivos conectados entre sí con el fin de compartir cualquier tipo de información, al hablar de dispositivos se incluyen una gran cantidad y variedad de dispositivos, computadores, impresoras, celulares, etc, partiendo de esa variedad la red se divide en varios subtipos, cuando se habla de redes LAN y WAN se relaciona directamente con las zonas geográficas, LAN (Local Area Network) es una red de computadoras que cubre un área geográfica pequeña, como una oficina, un edificio o un campus universitario, al contrario de WAN (Wide Area Network) es una red de computadoras que cubre un área geográfica amplia, como una ciudad, un país o incluso el mundo. 
 Un internetwork es una red formada por la interconexión de varias redes. Por ejemplo,al interconectar redes las LAN mediante una WAN obtendremos una red internetwork permiten compartir recursos y comunicarse entre sí de manera eficiente, y si unimos redes internetwork, tanto públicas como privadas, y aplicamos algún protocolo en su mayoría tcp/ip surge el concepto de Internet.

3. ¿Cuál es la diferencia entre INTERNETWORK E INTERNET?
4. Describir la diferencia entre Tarjeta de interfaz de red (NIC), puerto físico e interfaz de red.

La tarjeta de interfaz de red (NIC) es un complemento hardware que se utiliza para conectarse a una red. El puerto físico es el lugar físico en un dispositivo de red donde se conecta el cable de red. Por último, la interfaz de red es el software que se utiliza para configurar y administrar la conexión de red del dispositivo.

5. ¿Por qué se dice que un protocolo es independiente de la tecnología? 
6. Consultar y explicar brevemente qué es una Unidad de Datos del Protocolo (PDU).

Estructura de datos usada para transferir información entre diferentes dispositivos. Una PDU contiene información de control y de usuario, incluyendo encabezados y datos de la capa de aplicación, capa de transporte, capa de red, capa de enlace de datos y capa física.
